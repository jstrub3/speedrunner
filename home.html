Speedrunner API Documentation

Version 0.2 - now with Sqlite3 database.

Supported GET URIs:

  • /games/titles/all - returns list of all game titles

return format:

[
"The Legend of Zelda: Ocarina of Time",
...
"Hollow Knight"
]

  • /games/categories/all - returns list of all game categories for all titles

return format:

[
"Inbounds",
...
"All Bosses"
]

  • /games/catgories/<game_title> - returns list of all categories for <game_title>

return format:

[
"Inbounds",

...
"Out of Bounds"
]

  • /speedruns/games/<game_title>/ - returns list of speedruns (player names and durations) for game_title arranged by category and sorted by fastest duration
    • Supports optional "/<count>" parameter to limit speedrun results

return format:

{
"Any%": [
{
"duration": "0:00:42",
"player": "S. P. Lunker"
},
...
{
"duration": "0:13:37",
"player": "ToadKing"
}
],
"Hell%": [
{
"duration": "0:02:54",
"player": "S. P. Lunker"
},
... 
{
"duration": "0:13:37",
"player": "ToadKing"
}
]
}

  • /speedruns/games/<game_title>/<category> - returns list of speedruns (player names and durations) for game_title and category sorted by fastest duration
    • Supports optional "/<count>" parameter to limit speedrun results

 return format:

[
{
"duration": "0:02:54",
"player": "S. P. Lunker"
},
...
{
"duration": "0:13:37",
"player": "ToadKing"
}
]

  • /speedruns/players/<player_name>/<category> - returns list of speedruns (game title, category, and durations) for <player_name> sorted by fastest duration
    • Supports optional "/<count>" parameter to limit speedrun results

 [
{
"category": "Any%",
"duration": "0:13:37",
"game_title": "Spelunky"
},
...
{
"category": "Any%",
"duration": "0:19:42",
"game_title": "The Legend of Zelda: Ocarina of Time"
}
]

Supported POST URIs:

All POST requests require header of "Content-Type: application/json"

  • /games/add/ - adds game title and optional categories OR updates an existing game title with additional categories in the format(s)
    • {"game_title":<title>, "categories":[<categories>]}
    • {"game_title":<title>}

 

  • /games/categories/add/ - adds a list of categories to a game title in the format
    • {"game_title":<title>, "categories":[<categories>]}

 

  • /speedruns/add/ - adds a speedrun in the format
    • {"game_title":<title>, "category":<category>, "player_name":<player_name>, "duration":<duration>}
